Usage Guidelines & Best Practices

Do: Color Usage

Use WIN Red (#D1262F) for primary actions, important highlights, and brand elements. Maintain sufficient contrast ratios for accessibility.

/* Primary button */ .btn-primary { background: #D1262F; color: #FFFFFF; }
Don't: Color Overuse

Avoid using WIN Red for large background areas or multiple elements simultaneously. This reduces its impact and can cause visual fatigue.

/* Avoid this */ .page-background { background: #D1262F; /* Too overwhelming */ }
Do: Typography Hierarchy

Follow the established typography scale for consistent information hierarchy. Use Inter font family for all text elements.

h1 { font-size: 48px; font-weight: 700; } h2 { font-size: 36px; font-weight: 600; } p { font-size: 16px; line-height: 1.6; }
Don't: Inconsistent Spacing

Avoid arbitrary spacing values. Always use the defined spacing scale (4px, 8px, 12px, 16px, 24px, 32px, 48px, 64px) for consistency.

/* Avoid random values */ .component { margin: 13px 7px 21px; /* Use scale instead */ }
Do: Component States

Always implement hover, focus, and disabled states for interactive elements to provide clear user feedback.

.btn:hover { transform: translateY(-1px); } .btn:focus { outline: 2px solid #D1262F; } .btn:disabled { opacity: 0.5; cursor: not-allowed; }
